# sshでEC2に接続する方法
コマンドプロンプトで```ssh -i "~/Downloads/```を入力し、AWSのマネジメントコンソールから接続したいEC2から「接続」をクリックし、「SSHクライアント」タブをクリックすると 例：```ssh -i "KrrXp2bKjU3chCVq.pem" ec2-user@ec2-44-212-7-64.compute-1.amazonaws.com``` と記載されているので```KrrXp2bKjU3chCVq.pem" ec2-user@ec2-44-212-7-64.compute-1.amazonaws.com```の部分だけコピーし、さっき入力した```ssh -i "~/Downloads/``とつなげた状態
